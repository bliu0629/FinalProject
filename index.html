CIS 130 Web Design

This course is designed for those individuals who have an interest in web development and introduces students to the fundamentals of creating and developing websites.

Why did I choose this CIS 130 Web Design Course?

I chose to take a web design course because I wanted to learn how websites are created. In today's digital world, websites are everywhere, and understanding how to build and design them feels like a valuable and practical skill that I might use in the future. I was especially interested in learning how to use HTML and CSS to create visually appealing and organized web pages.
