Content Details for Each Page
Page 1: The title page will consist of the title of this final project, a description of why I chose this course. The navigation bar will be included to help users navigate through the webpage. Also, a link to my GitHub profile will be included. 
Page 2:  This page will explain how Visual Studio Code is used to organize html, css styling, images, and other files. This page will also include a video briefly explaining how Visual Studio Code is used. 
Page 3: This page will explain the meaning of HTML and example the usage of semantic tags such as main, nav, and footer. This page will also include a youtube video that shows how to use html code and a link to a website that does extra explanation on html concepts. 
Page 4:  This page will explain common layouts such as Grid and Flexbox. It will contain a description of how CSS styling helps define how the website is presented such as colors, fonts, and positioning. A table will be included to compare Flexbox vs. Grid layout. This page will also include a link to a website that helps explain CSS layout and styling more in depth. A youtube video will also be provided to explain more about CSS concepts. 
Page 5: This page will explain how important it is for a website to be accessible for impaired individuals. It will show how alt text, color contrast, animations, and keyboard navigation helps people with disabilities access the website. It will include a youtube video explaining more about websites that are accessible vs. websites that are not accessible. 
Page 6: This page will describe ethical issues such as giving credit to the original creators of the media used on your webpage and explain why web developers should be cautious about what they included on their webpage. This page will include a youtube video explaining deeper about ethical issues and a link to an article that talks more about ethical issues that occur in web development. 
Page 7: This page will describe how important it is to check on your codes and validate them so that there are no errors when the website is published. This page will include a video where it explains the importance of code validation and also website links to HTML and CSS code validators. 
Page 8: This page will explain the usage of artificial intelligence in your work. Artificial intelligence can be useful to brainstorm ideas for the webpage and debug errors in the code. This page will also talk about how you should avoid copying straight from AI generated codes and always use your own code. This page will include a video about using AI for web development and a link to an article that talks about how AI has affected web development.
